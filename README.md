# Kanban Board

A drag-and-drop task management board built with vanilla HTML, CSS, and JavaScript.

## Features
- Three columns: To Do, In Progress, Done
- Drag cards between columns using the HTML5 Drag and Drop API
- Add new cards per column (Enter to submit, Escape to cancel)
- Delete cards with the × button
- Progress bar tracking completion ratio
- All data persisted automatically in localStorage

## Usage
Open `index.html` in any browser. No install or build step needed.
